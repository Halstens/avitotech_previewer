## **Сервис назначения ревьюеров для Pull Request’ов (Avito)**

Cервис, который назначает ревьюеров на PR из команды автора, позволяет выполнять переназначение ревьюверов и получать список PR’ов, назначенных конкретному пользователю, а также управлять командами и активностью пользователей.


#### Использование приложения
1. Клонируете репозиторий командой `git clone <ссылка_на_проект>`
2. Запуск в Bash с помощью Makefile или docker-compose
   - make docker-up \ docker-compose up --build
3. Остановка сервисов
   - make docker-down -> make clean \ docker-compose down -v
4. Сервер развернется сразу на порту 8080


####Технологии
`Go 1.21` `PostgreSQL 15` `golang-migrate` `Docker + Docker Compose`
